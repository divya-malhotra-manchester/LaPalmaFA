# LaPalmaFA
Geometric morphometric analysis of fluctuating asymmetry and photosynthetic efficiency in Ageratina adenophora, Caldera de Taburiente National Park, La Palma (2026)
This repository contains all data and code for an analysis of fluctuating 
asymmetry (FA) and chlorophyll fluorescence (Fv/Fm) in the invasive species 
Ageratina adenophora, sampled across canopy positions in Barranco de las 
Angustias, Caldera de Taburiente National Park, La Palma, January–February 2026.

Contents:
- Rep1_Data.csv / Rep2_Data.csv — ImageJ landmark coordinate data (replicates 1 & 2)
- FvFm_R.csv — Chlorophyll fluorescence data
- La_Palma_FA_vs_FvFm.R — Full R analysis pipeline
- /outputs — Statistical output files generated by the script

Requirements: R (≥ 4.0), packages: geomorph, tidyverse, Hmisc

To replicate: download all files to one directory, open the .R script in 
RStudio, set working directory to source file location, and click Source.
